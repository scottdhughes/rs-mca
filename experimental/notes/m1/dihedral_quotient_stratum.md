# Dihedral Quotient Stratum

Status: PROVED.

Source DAG node: `dihedral_quotient_stratum`.

## Statement

For

```text
P_m = { p : p(X) = X^{2m} p(X^{-1}) },
```

the flat has dimension `m+1` and can be written as

```text
P_m = X^m F[X + X^{-1}]_{<=m}.
```

It is gcd-trivial on a multiplicative domain, and it is not contained in any
nontrivial multiplicative pullback `X^e g(X^M)`. Nevertheless, inverse-pair
orbits give many sparse closed sets: after removing the fixed points `+1` and
`-1`, every selected collection of at most `m` inverse-pair orbits contributes
a closed set, so the sparse-support closed-set lattice has size at least

```text
sum_{h<=m} binom((n-2)/2, h).
```

Thus the quotient taxonomy must include the dihedral/Chebyshev class

```text
X^e g(X^M + X^{-M}).
```

## Proof

The expressions

```text
X^m (X + X^{-1})^i, 0 <= i <= m,
```

span exactly the self-reciprocal degree-`2m` polynomials. They are independent
because the largest exponent in the `i`th expression is `m+i`, which appears
first there. Hence `dim P_m = m+1`.

The flat is gcd-trivial on a multiplicative domain because `X^m` belongs to
`P_m` and has no zero on the domain. It is not contained in a nontrivial
pullback `X^e g(X^M)`, since its monomial supports include adjacent exponents,
forcing any common residue class modulo `M` to have `M=1`.

For each inverse pair `{a, a^{-1}}`, every `p in P_m` satisfies

```text
p(a) = a^{2m} p(a^{-1}).
```

Thus the two trace coordinates are linearly dependent, giving one weight-two
dual word per inverse pair. The Chebyshev coordinate `y = X + X^{-1}`
identifies inverse-pair orbits, and evaluation on those orbits is a nonzero
diagonal rescaling of degree-`<=m` Reed-Solomon evaluation in `y`.

Choosing `h <= m` inverse-pair orbits gives a closed set generated by those
weight-two dependencies. Therefore the closed-set lattice has at least
`sum_{h<=m} binom((n-2)/2, h)` members. For growing `m` this is
super-polynomial, while for fixed dimension `d=m+1` it remains bounded by
`n^O(d)`.

## Non-Claims

This packet identifies the dihedral/Chebyshev quotient stratum and its closed
sets. It does not prove the broader many-sparse classification theorem, and it
does not close any row-level Hankel safe-side certificate by itself.
