# Support-Lattice Descent Accounting Packet

- **Status:** PROVED / accounting identity.
- **DAG node:** `f_support_lattice`.
- **Certificate:** `experimental/data/certificates/f-support-lattice/f_support_lattice.json`.
- **Verifier:** `python3 experimental/scripts/verify_f_support_lattice_packet.py --check experimental/data/certificates/f-support-lattice/f_support_lattice.json`.

This packet packages the closed-set accounting identity used by the Hankel
termination route. It is not a new uniform polynomial bound; it identifies the
finite combinatorial object that later family-specific arguments must count.

## Claim

In the sparse-dual descent, each branch is indexed by a sparse support. The
state reached by a branch is determined by the closed set generated by the
supports already chosen, together with the residual flat and residual degree
data. Once two branches have the same closed-set state, their future recursive
calls are identical.

Consequently the naive binary descent tree memoizes to a directed walk on the
lattice of closed support sets generated by the sparse supports. Along a strict
chain, either the flat dimension drops or the locator degree-drop budget drops.
Thus each strict chain has length bounded by the initial dimension plus the
initial degree-drop budget, and the memoized tree size is controlled by the
number of generated closed sets, up to the recorded bounded weight-label
factor.

## Boundary

The packet proves the accounting identity:

```text
descent tree size <= closed-set count * (1 + W_max).
```

It does not prove that every family has polynomially many closed sets. That
counting step is deliberately left to family-specific termination nodes such
as `hankel_rank_profile_entropy` and `f_termination_hankel`.

## DAG Use

`hankel_rank_profile_entropy` consumes this packet to replace binary branch
growth by entropy over the reachable closed-set lattice. This is the point
where the Hankel route stops trying to count raw descent branches.

## Non-Claims

- This packet does not prove a uniform polynomial closed-set bound.
- This packet does not classify which sparse supports occur in a given family.
- This packet does not close the Hankel termination theorem by itself.
- This packet does not edit Papers A-D.
