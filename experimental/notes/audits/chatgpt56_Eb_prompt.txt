GOAL (Conjecture E-b). N <= n^3. Equivalently, with nu(v) = #{|S|=m : (p_1(S),...,p_w(S)) = v} for v in F_p^w (so N = nu(0)): the rotation-fixed value nu(0) is not atypically large, i.e. nu(0) <= poly(n) * mu.

VERIFIED STRUCTURE (established; please still check):
(A) Li-Wan cycle expansion. With pi(c) = sum_{a in mu_n} e_p( c . (a^j)_{j=1..w} ), c in F_p^w:
    N = p^{-w} sum_c pi(c)^m = sum_{partitions lambda of m} ((-1)^{m-l(lambda)}/z_lambda) M_lambda,
    M_lambda = p^{-w} sum_c prod_r pi(rc)^{m_r}   (m_r = multiplicity of part r, l = #parts).
    Low-cycle vanishing: M_lambda = 0 if l(lambda) <= w (Vandermonde in distinct values). So ONLY partitions with > w cycles contribute.
(B) Descent. A mu_2-symmetric S = -S bijects (via u = t^2, the 2:1 map mu_n -> mu_{n/2}) with U subset mu_{n/2}, |U| = m/2, p_i(U) = 0 for i <= floor(w/2); hence N_prim = N - 1[2|m] N(n/2, m/2, floor(w/2)) is the primitive (free-mu_2-orbit) count.
(C) Rotation. p_j(zeta S) = zeta^j p_j(S), so v=0 is rotation-FIXED while every v != 0 sits in a size-n orbit of equal nu; thus N_prim = n * (#free mu_n-orbits), and the additive energy E = sum_v nu(v)^2 = nu(0)^2 + n * sum_{v != 0 reps} nu(v)^2.
(D) p-sensitivity. For SMALL p at the same (n,m,w), N >> n^3; the conjecture holds only because deployed mu << n^3. So a proof MUST use mu small (p large); a p-blind structural bound is impossible.
(E) Energy is near-random. E ~ C(n,m)^2/p^w. Its tail E - C(n,m) = sum_{d>w} C(n-2d,m-d) E_d, with E_d = #{disjoint A,B in mu_n, |A|=|B|=d, p_j(A)=p_j(B) for j<=w}; E_d = 0 for d <= w (PTE rigidity, Newton); the tail is bulk-dominated at d ~ 0.1-0.25 n.
