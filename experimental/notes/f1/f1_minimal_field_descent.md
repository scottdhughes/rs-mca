# F1 Minimal Field Descent

Status: PROVED.

Source DAG node: `f1_minimal_field_descent`.

## Statement

Let `B = F_p(D)` be the field generated by the domain data and let
`F = F_{p^N}` be the ambient line/sampling field, with `B <= F`.  For an
extension-line bad slope produced by a pair `(u,v)`, package the associated
syndrome-pencil coefficients and slope as projective pencil data `x` over
`F`, modulo the usual common scaling ambiguity.

Then `x` has a unique minimal field of definition `K` with

```text
B <= K <= F.
```

Consequently the S6/F1 case split is exhaustive and mutually exclusive:

```text
K = B,        B < K < F,        K = F.
```

These cases are respectively the base-rational, intermediate-tower, and
genuinely `F`-valued branches.

## Proof

Write `F = F_{p^N}`.  Its subfields are exactly `F_{p^d}` with `d | N`, and
the subfield lattice is the divisor lattice:

```text
F_{p^a} cap F_{p^b} = F_{p^gcd(a,b)},
F_{p^a} F_{p^b}     = F_{p^lcm(a,b)}.
```

The Galois group `Gal(F/F_p)` is cyclic, generated by Frobenius.

Choose homogeneous coordinates for the projective pencil datum

```text
x = [a_0 : ... : a_m].
```

Define

```text
Stab(x) = {sigma in Gal(F/F_p) : sigma(x) = x as a projective point}.
```

Equivalently, after choosing any nonzero coordinate `a_j`, this stabilizer
fixes every ratio `a_i/a_j`.  It is a subgroup of the cyclic Galois group.
Set

```text
K_0 = F^{Stab(x)}.
```

The ratios `a_i/a_j` are fixed by `Stab(x)`, hence lie in `K_0`, so `x` is
defined over `K_0`.  Conversely, if `x` is defined over an intermediate field
`K'`, then every element of `Gal(F/K')` fixes the same projective ratios.
Thus

```text
Gal(F/K') <= Stab(x).
```

By Galois correspondence this gives

```text
K_0 = F^{Stab(x)} <= F^{Gal(F/K')} = K'.
```

Therefore `K_0` is contained in every field of definition of `x` and is itself
a field of definition.  It is the unique minimal field of definition over
`F_p`; the composite-ambiguity falsifier cannot occur in this finite-field
setting.

Now write `B = F_{p^b}`.  The minimal field of definition containing the base
field is

```text
K = B K_0.
```

If `K_0 = F_{p^d}`, then by the divisor-lattice formula

```text
K = F_{p^lcm(b,d)}.
```

This is the unique smallest subfield of `F` containing both `B` and the
minimal field over `F_p`.  Since `K` lies in the interval `[B,F]`, exactly one
of the three alternatives holds:

```text
K = B,
B < K < F,
K = F.
```

This proves the F1 minimal-field descent and the exhaustiveness of the tower
case split.

## Program Use

This is field-theoretic bookkeeping for the extension-line MCA fork.  It does
not prove the hard pole-forcing/classification theorem.  It only guarantees
that every extension-line bad-slope datum has a well-defined minimal field,
so the ledger can route the datum into base-rational, intermediate-tower, or
genuinely full-field branches without a fourth "composite ambiguity" case.
