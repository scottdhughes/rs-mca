# XR Light-Profile Eliminant Nonvanishing

This directory stores the lightweight replay certificate for
`experimental/notes/m1/xr_light_profile_eliminant_nonvanishing.md`.

Replay:

```bash
python3 experimental/scripts/verify_xr_light_profile_eliminant_nonvanishing.py --emit
```

The JSON certificate records a finite enumeration of small Venn-profile light
triangles and verifies that the normal-form matrix has full column rank `3t`.
The finite replay is a schema guard; the note contains the parametric proof.
